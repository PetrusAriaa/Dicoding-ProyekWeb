Belajar Golang - Perulangan

Perulangan pada pemrograman adalah logika yang dilakukan dengan eksekusi blok kode secara berulang selama memenuhi kondisi tertentu.

Berbeda dengan bahasa pemrograman lain seperti JavaScript atau C++ yang memiliki perulangan for dan while, Golang hanya memiliki perulangan for. Meski demikian for pada Golang memiliki kemampuan sebagaimana while dan forEach.

Perulangan for

Cara pertama dengan memasukkan variabel counter, kondisi, dan iteratornya. Coba perhatikan dan praktekan kode berikut. 

for i := 0; i < 5; i++ {
  fmt.Println("counter =", i)
}

Kode tersebut akan menjalankan perulangan dengan titik awal i bernilai 0. Jika nilai i lebih kecil 5, kode fmt.Println("counter =", i) akan dijalankan. Setelah itu nilai i akan ditambahkan 1. Begitu seterusnya hingga i tidak lagi memenuhi kondisi i < 5.

Perulangan for dengan Argumen Hanya Kondisi

Cara berikutnya adalah dengan menuliskan kondisi setelah keyword for tanpa inisiasi variabel counter terlebih dahulu. Cara ini merupakan bentuk while 

i := 0
for i < 3 {
  fmt.Println("Halo dunia!")
  i++
}

Perulangan for tanpa Argumen

Kemudian adalah dengan menuliskan for tanpa kondisi yang membatasi penjalanan blok kode di bawahnya.

Bentuk ini merupakan cara Golang membuat infinite loop seperti while True. 

i := 0
for {
  fmt.Println("counter =", i)
  i++
}
